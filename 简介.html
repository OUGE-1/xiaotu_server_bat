📌 项目概述

xiaotu_server.py 是一款基于 Python 标准库和 Tkinter 图形界面框架开发的 Minecraft 服务器图形化管理工具。它通过可视化的操作面板，帮助服主轻松完成服务器的启动/停止、控制台交互、玩家管理、快捷命令配置及趣味互动等操作。

该工具的设计初衷是降低服务器管理门槛，让即使不熟悉命令行操作的玩家，也能像使用桌面软件一样便捷地运维自己的 Minecraft 服务器。它仅依赖 Python 3.6+ 和 Java 8+ 环境，无需安装任何第三方库，真正做到开箱即用、绿色便携。

🧩 完整功能介绍

1. 服务器生命周期管理（核心）

功能说明
启动服务器通过文件对话框选择服务端 JAR 文件（如 server.jar、paper.jar 等），设置 -Xms（初始内存）和 -Xmx（最大内存）后一键启动。
停止服务器一键优雅终止服务端进程，避免数据损坏。
状态监控底部状态栏实时显示"未启动 / 启动中 / 运行中"三种状态。

2. 交互式控制台

  • 实时日志输出：服务器运行时的所有日志（玩家进出、系统消息、报错信息）实时滚动显示，并自动追随最新一行。
  • 命令输入：在下方的输入框中输入任意 Minecraft 命令（如 /op 玩家名、/gamemode creative），点击"发送"或按回车键即可发送。服务器未启动时输入框自动禁用，防止误操作。

3. 快捷命令系统

  • 通过编辑同目录下的 quick_commands.json 文件，可自定义一组常用命令按钮。
  • 示例配置：
    { "保存世界": "save-all", "在线列表": "list", "欢迎消息": "say 欢迎来到服务器！" }
  • 程序启动时自动加载并生成按钮，点击即可一键发送命令。若 JSON 文件格式错误或缺失，程序会优雅忽略并继续运行。

4. 玩家管理面板

  • 在线玩家列表：自动解析 list 命令输出，每 5 秒动态刷新，实时反映玩家进出情况。
  • 右键快捷操作：在玩家名称上右键点击，弹出菜单支持快速复制玩家名到剪贴板。

5. 玩具互动面板（特色彩蛋功能）

  • 预定义了一批 Minecraft 特色物品（如钻石剑、三叉戟、弩、盾牌、烟花火箭等）。
  • 操作流程：选中一名在线玩家 → 选中一个玩具 → 点击"赠送"按钮 → 程序自动构造并发送 /say 服务器赠送 XXX 一个[玩具名称]！ 消息到游戏内聊天栏。
  • 适用于趣味互动、小游戏奖励、欢迎新人等场景。

6. server.properties 配置编辑器

  • 内置图形化的 server.properties 文件编辑器，无需手动查找文件。
  • 支持修改 max-players、motd、gamemode、difficulty 等全部服务端参数，修改后可直接保存。

7. 辅助功能

  • 跨平台支持：使用 pathlib.Path 处理路径，兼容 Windows / Linux / macOS。
  • 窗口关闭保护：关闭主窗口时自动终止仍在运行的服务器进程，防止后台残留。
  • 异常容错：对文件读写、子进程管理、JSON 解析等关键操作均做了异常捕获，避免因小错误导致程序闪退。

📌 Windows 7 特别说明

⚠️ 关键兼容性要点：

  • 系统补丁：必须安装 Windows 7 Service Pack 1 (SP1)。
  • Python 版本：务必使用 Python 3.8.x（最新小版本），不支持 3.9 及更高版本。
  • Java 版本：推荐 Java 8 Update 20 (8u20) 或更新版本。
  • 其他：安装路径避免包含空格和中文字符；安装 Python 时勾选 "Add Python to PATH"。

🎯 总结

xiaotu_server.py 从最初基础的启动器，逐步演变为集图形化控制台、快捷命令面板、玩家管理、玩具彩蛋互动、配置文件编辑于一体的综合性服务器管理工具。开发过程中曾尝试集成 UUID 查询功能，但最终为确保稳定性和易用性，选择移除了这一依赖外部 API 的模块。

该工具特别适合：

  • 新手服主快速上手管理服务器
  • 小型私人服务器日常运维
  • 与朋友联机时进行趣味互动

其"仅依赖 Python 标准库 + Java 运行环境"的轻量设计，使其在 Windows 7/10/11、Linux、macOS 等主流平台上均可轻松部署运行。
